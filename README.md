## 微信扫码登陆
- 最近研究观察了微信扫码登陆相关内容，做一下分享
- 我们如果要登陆微信，会有一个二维码，其实一般二维码就是里面包含字符串信息的一张图片，我们解析微信二维码，可以看到
```text
http：//login.wexin.qq.com/l/obsbsQ-Dzk==
```
- 这样后面的一串字符如何实现登陆呢? 这其实讲述的是一个思路，在自行设计二维码登陆的时候可以用上
1) 在展示这个二维码的时候，就是微信服务器向客户端发送了信息，生成了一个全局唯一的ID
2) 手机扫描后，前提是手机微信是登陆状态，那就能知道手机用户相关信息，通过扫描能够拿到二维码内部的字符串ID，将这两部分数据提交到微信服务器
3) 如果服务器通过了ID和用户的绑定认证，就给页面返回该用户的相关信息
4) 其中网页还会后微信服务器后端建立一个长连接，会不断轮询二维码扫描情况，主要是通过字段标识，如果扫描成功返回201，则等待接收用户信息，如果超时就进行下一次请求，即申请新的全局页面ID，需要用户再次扫描授权
5) 微信服务器除了给网页发送用户的相关信息，还会给页面回一个身份认证的token,需要在下一次请求中带上
